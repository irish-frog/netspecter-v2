# NetSpecter Sophos SX/SG 125 Rev.3 Hardware Parts

This folder contains STL files for recreating the pictured NetSpecter rack
brackets and LCD back cover.

## Fitment

These parts are intended for Sophos SX/SG 125 Rev.3 appliances.

They may also fit Sophos SX/SG 135 Rev.3 hardware, but that fitment is not yet
confirmed. Check measurements before printing or installing.

## Files

| File | Purpose |
|---|---|
| `left-hand-side.stl` | Left rack bracket / LCD side mount |
| `right-hand-side.stl` | Right rack bracket |
| `lcd-back-cover.stl` | Back cover for the round LCD display |

Print and install at your own risk. Verify clearances, airflow, screw length,
and display wiring before mounting the appliance in a rack.
